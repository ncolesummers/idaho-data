# Internal Package

Private application code that is not intended to be imported by other projects.

## Structure

- `cache/` - Caching layer implementation
- `config/` - Configuration loading and validation
- `data/` - Data access layer and repositories
- `database/` - Database connection and migrations
- `events/` - Event handling and pub/sub
- `handlers/` - HTTP request handlers
- `metrics/` - Application metrics and monitoring
- `models/` - Data models and database schemas
- `services/` - Business logic and data processing
